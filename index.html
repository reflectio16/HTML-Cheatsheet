HTML Image and Multimedia References

Image and Multimedia Tags

Tag Name Description

<area>

 Area Defines an area inside an image map that has predefined clickable areas.

<audio>

 Audio Used to embed sound content in documents.

<img>

 Image Embeds an image into the document.

<map>

 Image Map Used with <area> elements to define an image map (a clickable link area).

<track>

 Track Used as a child of the media elements, audio and video.

<video>

 Video Embeds a media player which supports video playback into the document.

Embedded Content

Tag Name Description

<embed>

 Embed Embeds external content at the specified point in the document.

<iframe>

 iframe A nested browsing context, embedding another HTML page into the current one.
